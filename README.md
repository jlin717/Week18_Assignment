My .NET 10 Web API & Console App Lab:

This repository contains my progress through this week’s hands-on .NET modules. I’m building a full solution from scratch that includes a Web API and a Console Application that communicates with it.
The goal of this project is to get comfortable with the "plumbing" of .NET applications and seeing how data moves from a server to a client.

Project Disclaimer:

I’m building this following a step-by-step guide designed for learning. While the apps are fully functional, the architecture is intentionally simplified for brevity. 
This isn't meant to be "production-ready" or "industry-standard" code quite yet and it’s a focused look at the core mechanics of C# and .NET 10.
